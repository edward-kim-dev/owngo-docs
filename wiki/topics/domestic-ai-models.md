## 모델별 전략적 활용

> [!analysis]
> OwnGo AI ERP 플랫폼 맥락 기준

### KT 믿음(Mi:dum) 2.0
- **장점**: 툴콜·리즈닝 능력, 128K 롱 콘텍스트, 2025년 7월 '호랑이 리더보드' 1위
- **OwnGo 활용**: ERP 실무형 에이전트 핵심 모델 — 자연어 명령 → gRPC 툴콜 변환

### LG EXAONE 4.1
- **장점**: MoE로 추론 시 10% 파라미터만 활성화(효율), '추론 모드' 지원
- **OwnGo 활용**: 복잡한 비즈니스 로직 추론, 글로벌 다국어 지원

### NCAI 바르코(VARCO)
- **장점**: 멀티모달(3D·사운드·비전), 연동 10분, 본선팀 무제한 크레딧
- **OwnGo 활용**: 대시보드 시각화, 리포트 생성 등 창의적 콘텐츠 요소

### SKT A.X K1
- **장점**: 519B 규모, 복잡한 실시간 추론·연산 특화
- **OwnGo 활용**: 고난도 아키텍처 설계 보조, 멀티스텝 업무 계획 수립

### 업스테이지 Solar Pro 3 & Document Parse
- **장점**: 한국어 최강, 비정형 문서(표·차트·손글씨) 마크다운 변환
- **OwnGo 활용**: RAG 파이프라인 문서 전처리, 재무/계약서 자동 분석

## 핵심 개념 용어 정리

> [!source]
> 원본 §비전공자를 위한 용어 설명

| 용어 | 설명 |
|------|------|
| **MoE** | 전문가 혼합 구조 — 전체 모델 중 필요한 부분만 활성화하여 효율 극대화 |
| **RAG** | 검색 증강 생성 — 외부 문서를 실시간 참조하여 환각(Hallucination) 감소 |
| **Tool-call** | AI가 날씨 확인, DB 조회 등 외부 도구를 직접 조작하는 기능 |
| **128K 롱 콘텍스트** | 한 번에 약 10만 단어(수백 페이지) 동시 이해 능력 |
| **Agentic** | 스스로 계획 수립 → 단계별 문제 해결하는 자율적 특성 |
