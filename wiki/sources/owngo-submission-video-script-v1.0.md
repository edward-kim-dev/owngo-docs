# AI ERP OwnGo 대회 제출 영상 통합 스크립트 v1.0

> [!source]
> 원본: `docs/[DOC] AI ERP OwnGo 대회 제출 영상 통합 스크립트_v1.0.md` (작성일: 2026-04-19, 작성자: 서명원)

## 문서 목적

인공지능 챔피언십 대회 제출용 3분 홍보 영상의 나레이션·자막·시각화 가이드를 포함한 통합 스크립트 v1.0. v0.8 대비 서론·팀원 소개·클로징 파트 대폭 수정.

---

## v1.0 주요 변경 사항

> [!analysis]
> v0.8 대비 주요 변경점 (원본 frontmatter `update` 필드 기준):
> - **서론**: 문구 수정 (자율적 실행 시스템, 비개발자 오픈소스 직접 수정 강조)
> - **팀원 소개**: 각 팀원 스크립트 수정 (직군 경계 없는 'All-round Solver' 콘셉트 강화)
> - **클로징**: 나레이션 문구 수정 ('AI 네이티브' 기준 제시)

## 파트별 핵심 내용

### 1. 서론 (00:00 ~ 00:30)

> [!source]
> 원본 §1

**작성 구조**: 질문(10~50인 중소기업 IT 인력 부재 지적, 10초) → 반전(자율적 실행 시스템으로의 혁신, 10초) → 선언(OwnGo + 팀 소개, 10초)

**핵심 메시지**: 중소기업은 IT 인력 부재로 오픈소스 자산화에 실패 → OwnGo는 비개발자도 자연어로 오픈소스 코드를 직접 수정·배포하는 AI 에이전트

**시각화**: 복잡한 엑셀/ERP 화면(Observe & Orient 텍스트) → OwnGo 채팅 인터페이스(Decide & Act 텍스트)로 전환 모션 그래픽

---

### 2. 팀원 소개 (00:30 ~ 01:15)

> [!source]
> 원본 §2

**공통 인트로**: *"OwnGo 팀의 직책은 단 하나, 올라운드 솔버(All-round Solver)입니다."* — 기획과 개발 경계 폐지, AI를 기본 작업 환경으로 삼아 비즈니스 문제 해결

#### 김광혁 — AI/Backend 담당
- **핵심 멘트**: "현장의 경험과 열정으로, AI와 기업 데이터의 안정적인 연결 고리를 구현합니다."
- **경력**: 5년 차, 빅데이터 플랫폼 고도화 및 오픈소스 개발
- **역할**: 전반적인 구조 기획 + MVP 개발 주도 (자연어 → 오픈소스 코드 수정·배포)
- **자막**: 빅데이터 플랫폼 고도화 및 오픈소스 개발 5년 차 | AI·Backend (gRPC 연동)

#### 선윤호 — Frontend/UX 담당
- **핵심 멘트**: "사용자의 시선에서, 누구나 대화하듯 쉬운 직관적인 인터페이스를 만들어갑니다."
- **경력**: 사내 AI 지식 통합 허브 PL + IT 교육 강사 (기획·개발·인프라 조율)
- **역할**: 사내 AI 지식 통합 허브·핵심 업무 시스템 구축, 비즈니스 요구사항 → 프롬프트·아키텍처 번역
- **자막**: 사내 AI 지식 허브 개발 PL & IT 교육 강사 | Frontend·UX (대화형 UI)

#### 서명원 — Infra/Architecture 담당
- **핵심 멘트**: "어떤 상황에서도 비즈니스가 멈추지 않는 단단한 시스템 기반을 설계합니다."
- **경력**: 12년 차, 데이터 아키텍처(DA) + CI/CD 구축 (공공·금융 대형 차세대 프로젝트)
- **역할**: AI 기반 온톨로지(Ontology) 설계로 비즈니스 확장성 창출 + 10~50인 조직용 안전한 인프라 구축
- **자막**: 12년 차 공공·금융 데이터 AA 및 CI/CD 구축 | Infra·Architecture (시스템 안정성)

---

### 3. 아이디어 및 데모 (01:15 ~ 02:35)

> [!source]
> 원본 §3

**3가지 시연 구성**:
- **Sync Overlay**: 사용자 입력 → gRPC 커넥터 → ERPNext API 1:1 매핑 오버레이
- **UX 변환**: 50+ 입력 필드 발주 폼 → AI 대화 한 줄 + Progressive Disclosure(점진적 공개) 애니메이션
- **기술 실체**: 화면 하단에 Terraform 코드 + 실시간 가드레일 작동 로그 시각화

> [!analysis]
> 데모 나레이션 포함 여부는 원본에서 검토 사항으로 명시됨 (원본 §3 상단).

---

### 4. 최종 클로징 (02:35 ~ 03:00)

> [!source]
> 원본 §4

**핵심 메시지**: OwnGo = **AI 네이티브(AI-Native)** 기업 운영의 새로운 기준. 자율적 비즈니스 사이클 구축.

**나레이션 키워드**: 레거시 탈피, IT 인력 없이도 시스템이 직접 문제 발견·실행, "기술적 신뢰를 비즈니스 가치로"

**자막**: `'Antifragility: 위기를 기회로 바꾸는 기술력'`, `'국산 AI 생태계의 비즈니스 가치 실현'`
